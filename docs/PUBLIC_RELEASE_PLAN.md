# Public release plan

1. Keep the full modified SOKE checkout private.
2. Publish this delta repository as the public artifact.
3. Point users to the official SOKE repository and pinned commit.
4. Let users apply `scripts/apply_delta.sh` locally.
5. Keep datasets/checkpoints outside Git and document how to obtain them.
6. If SOKE authors grant permission for derivative redistribution, consider a
   later full-source release.
